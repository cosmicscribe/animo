# 🎬 animo — Motion Design Portfolio

A modern motion design portfolio showcasing creative animations, visual storytelling, and interactive experiences.

Built with Vite + TypeScript for high performance and smooth user experience.

👉https://animo-bikn.vercel.app/

## 🎨 About

animo is a motion design-focused portfolio that highlights:

- Motion graphics  
- Animation projects  
- Creative concepts  
- Professional work showcase  

Designed to deliver a visually engaging and smooth browsing experience.

├── public/          # Static assets (images, videos, icons)  
├── src/             # Main application code

## 📦 Installation & Setup

Clone the repository:

git clone https://github.com/your-username/animo.git  
cd animo  

Install dependencies:

npm install  

Run development server:

npm run dev  

Build for production:

npm run build  

Preview production build:

npm run preview  

---

## 🛠️ Customization

- Update content inside /src  
- Replace media files in /public  
- Add your own projects and animations  
- Adjust styles and layout as needed
